Installation & Désinstallation

Accueil Bestiaire Crew Spécificités Installation/desinstallation

Pour installer le jeu, téléchargez le en cliquant sur le lien et lancez le logiciel d'installation. Pour désinstaller le jeu, il vous suffit de supprimer le jeu téléchargé ainsi que la sauvegarde se trouvant dans 

C:\Users\[UserName]\AppData\LocalLow\Les Canailloux\Fauteur de troubles

Télécharger le setup (Windows x64)

Installation

Etant donné que nous sommes de vils h@ck3rs, Windows ne nous connaît pas. Il faut cliquer sur "Informations complémentaires".

Cliquez ensuite sur "Exécuter quand même".

Executez le programme d'installation.

Choisissez le répertoire où vous souhaitez installer le jeu.

Choisissez si vous voulez ou non un raccourci sur le bureau.

Cliquez sur "Installer".

Bravo, l'installation est terminée ! Vous pouvez maintenant tester le jeu !

ATTENTION : lorsque vous lancerez une partie pour la première fois, il faudra autoriser l'accès au niveau du pare-feu.
Si vous souhaitez rejoindre une partie (et que vous n'en avez jamais lancé / rejoint), vous devez impérativement avoir aussi autorisé l'accès au préalable.
Il faut donc lancer une partie la première fois que vous jouez, ce qui fait apparaître cette fenêtre qui vous permet d'autoriser l'accès.

Cette ligne est inutile, ce qu'il faut faire est écrit plus gros au dessus.

Désinstallation

Rendez-vous dans le répertoire où vous avez installé le jeu, puis lancez "unins000.exe"

Cliquez sur "Oui".

Cliquez sur "Ok".

ATTENTION : Il faut aussi supprimer la sauvegarde

Elle se trouve dans 

C:\Users\[UserName]\AppData\LocalLow\Les Canailloux\Fauteur de troubles

Vous pouvez supprimer le dossier "Les Canailloux" entièrement.
